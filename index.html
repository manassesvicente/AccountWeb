Introduzindo Account Web

Precisa de razões para amar o Account Web? Olhe adiante.

Por contadores, para contadores

Construído por @manassesvicente, o AccountWeb utiliza Scriptcase, com banco de dados Firebird, e é gerenciado pelo GitHub para ajudar os contadores iniciantes a aprender ciências contábeis

Mobilidade

Account Web foi feito não somente para ser fácil e comportar bem nos últimos browsers para desktop (até o IE7!), mas em tablet e browsers do smartphone via CSS responsivo também.

Funcionalidades

As funcinalidades desde formulários simples, e até Relatórios Personalizáveis com vários formatos de impressão fazem do Account Web um ótimo sistema de ensino.

O que é necessário

Para instalar o AccountWeb, visite o o site Manassés Vicente ou siga o manual com exemplos.

Universidades usando o Account Web


$ cd manassesvicente/accountweb
$ git fetch origin
$ git checkout gh-pages
